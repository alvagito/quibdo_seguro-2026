## 📋 **Próximos Pasos**

### **1. Clonar en Producción**
```bash
git clone https://gitlab.com/alvarochaverra0510/quibdo_seguro_final.git
cd quibdo_seguro_final
```

### **2. Configurar Entorno**
```bash
composer install
cp .env.example .env
php artisan key:generate
```

### **3. Configurar Base de Datos**
- Instalar MongoDB
- Configurar conexión en `.env`
- Ejecutar seeders opcionales

### **4. Desplegar**
- Configurar servidor web
- Establecer permisos de archivos
- Configurar SSL (recomendado)
